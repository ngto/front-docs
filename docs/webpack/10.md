# webpack + Vue

## 1. 创建项目文件夹(webpack-vue)

## 2. 控制台进入项目文件夹生成 package.json 文件

```dotnetcli
npm init -y
```

## 3. 使用 webpack 先要安装 webpack 和 webpack-cli

- -D 是 --save-dev 的缩写

```dotnetcli
yarn add webpack webpack-cli -D
or
npm install webpack webpack-cli -D
```
