- 总结
  > - 作用域是一套规则，用于确定在何处以及如何查找变量，如果查找的目的是对变量赋值，那么就使用 LHS 查询，如果目的是获取变量的值，就用 RHS 查询，
  > - 全局作用域(执行上下文都能访问到，window)
  > - 函数作用域(函数内能访问，funciton)
  > - 块级作用域(es6)通过 let 和 const 实现块级作用域
