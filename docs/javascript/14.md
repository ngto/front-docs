# 手写 instanceOf

- 1. 在 left 的原型链中层层查找，是否有原型等于 prototype
- 2. 确定边界条件，如果 left === null，即找到头没找到返回 false，right === left，即找到返回 true
- 3. left = left.proto，不停的向上查找

```
const myInstanceof = function(left, right) {
    right = right.prototype;
    left = left.__proto__;
    while (true) {
        if (left === null) {
            return false;
        }
        if (right === left) {
            return true;
        }
        left = left.__proto__;
    }
}
```
